## ⚠️ Yasal Uyarı
Bu dosyalar **sadece aşağıdaki amaçlarla kullanılabilir**:

- Güvenlik araştırmaları


# 1. Yeni repo oluştur

# 2. README ve PDF ekle

wget https://example.com/sample.pdf

# 3. GitHub'a yükle
